## Brownie
This is a canvas toolkit in coffeescript, it can be used to build a static image/text designer.

## Features
- [x] support high-dpi
- [x] support layers - only one canvas is used
- [x] support absolute/relative position
- [x] support z-index
- [x] support hit test
- [x] keyboard util
- [ ] image layer
- [ ] text layer
